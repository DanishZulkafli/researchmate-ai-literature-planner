# ResearchMate AI — Literature Review Planner

ResearchMate AI is a smart research planner and literature review matrix for students, researchers, postgraduate candidates, and PhD users. It helps users organize research papers, generate research plans, track literature review progress, identify research gaps, and export literature data.

This project is fully static and can be deployed directly using GitHub Pages.

## Live Demo

Live project: https://danishzulkafli.github.io/researchmate-ai-literature-planner/

## Features

- AI-style research plan generator
- Research topic and field selection
- Research goal and timeframe selection
- Suggested search keywords
- Suggested research questions
- Suggested methodology
- Research timeline generation

- Track paper title, authors, year, and source type
- Track methodology type
- Track relevance score
- Track quality score
- Track reading status
- Track keywords
- Track key findings
- Track research gaps and limitations
- Optional paper link
- Literature review matrix
- Search papers

- Filter by reading status

- Research score calculation
- Methodology breakdown

This app does not use a real AI API. The AI-style suggestions are generated using rule-based logic in JavaScript. It is suitable for portfolio, demo, and personal research planning use.
